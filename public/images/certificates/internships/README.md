# Internship Certificates

Place your internship certificate images in this directory with the following exact file names:

1. `ibm-internship.jpg` - IBM SkillBuild Internship Certificate (Edunet Foundation/AICTE)
2. `mahat-internship.jpg` - Mahat Labs Private Limited Internship Certificate
3. `gateway-internship.jpg` - Gateway Software Solutions Internship Certificate

## Notes:
- All images should be in JPG format
- Recommended size: 800x600px or similar aspect ratio
- File names must match exactly (case-sensitive)
- If an image is missing, a placeholder will be shown
